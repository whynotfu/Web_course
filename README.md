# Web Courses

Репозиторий с двумя курсами по вебу:

- **Основы веб-технологий** — 10 лабораторных
- **Проектирование веб-сервисов** — материалы/лабы по 2-му предмету

## Структура

- `web-basics/` — основы веб-технологий (10 лаб)
  - `lab1/` … `lb10/`
- `web-services-design/` — проектирование веб-сервисов

## Как пользоваться

Заходи в папку предмета → выбирай нужную лабу/тему → в index.html сразу перечень всех лаб.
